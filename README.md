## 사용권한
#### 터미널 내에서 "ls -l"할 경우 "drwx"확인가능
#### 이때 각 "drxw"는 
##
#### d - directory
#### r - read permission
#### w - write permission
#### x - execute permission
##
#### 위와 같이 의미한다.
##
#### 또한 터미널 창에서 조회하고자하는 권한의 폴더에서 아래와 같이 예시가 있다고 하자 이때각 위치는 아래와 같이 구분이 된다.
#### drwx/r--/w--/x-- 
#### d:폴더/user/group/other
