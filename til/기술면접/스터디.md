this는 JS-Engine 에서..
1. 객체(변수) [call stack] var 
2. scope [chained..]
3. this [context] : global (window) <-- binding(let으로 변수를 지정할 경우 binding이 안됨)
4. 값을 할당
5. hoisting
```js
var a = 10;
const test = () => {
    console.log(this.a)
}

let obj = {
    value : 11,
    func : function () {
        console.log(this)
    }
}

let temp = obj.func
temp() // this가 global이 되므로 undefined
// 추가로 봐야할 내용 (call, apply, new)
```
